# Code of Conduct

This project adopts the Contributor Covenant, version 2.1, as its
code of conduct.

The full text of the Contributor Covenant 2.1 is published at:

  https://www.contributor-covenant.org/version/2/1/code_of_conduct/

By participating in this project (filing issues, opening pull
requests, commenting, or any other form of contribution) you agree
to abide by its terms.

## Reporting

If you observe behavior that violates the Contributor Covenant,
please report it through one of the channels listed in
[`SECURITY.md`](SECURITY.md) (the same private channels used for
security disclosures). All reports will be handled discreetly.

The project maintainer is responsible for enforcement and will
respond as outlined in the Contributor Covenant's enforcement
guidelines.
